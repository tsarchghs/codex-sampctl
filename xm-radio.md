# XM Radio (GTA:World Feature Showcase)

## Source

GTA World Forums - GTA V Heavy Roleplay Server

## Overview

Nightlife is a major part of the roleplay scene in GTA:World, and clubs are a regular activity for many players. The XM Radio system builds on earlier implementations from GT-MP with a refined feature set. In addition to open audio formats like OGG, XM Radio supports YouTube Live video streams as radio stations, expanding the pool of in-game stations available to players.

## Features

- XM Radio system fully working and synced in properties and vehicles.
- Audio stream support for open formats (Vorbis OGG, AAC, etc.).
- Friendly GUI for admins to add new stations for players without developer involvement.
- Volume controls and metadata display in-game.

## Commands

- `/setstation [1-100]` (inside a house or property)
- `/radioshow`
- `/radiohide`
